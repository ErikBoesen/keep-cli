keep-cli
========

Google Keep frontend for terminals.

Setup
-----

- Copy `config.example.yml` to `config.yml`.
- Modify as necessary
- Run `./keep`

Keys
----

- <kbd>k</kbd> Up
- <kbd>j</kbd> Down
- <kbd>h</kbd> Left
- <kbd>l</kbd> Right
- <kbd>f</kbd> Pin/Unpin
- <kbd>e</kbd> Archive/Unarchive
- <kbd>r</kbd> Sync
